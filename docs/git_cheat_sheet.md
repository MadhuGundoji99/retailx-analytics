## Git Commands (cheat-sheet)
Action	Command	Meaning
Check status	git status	Shows changed/untracked files
Add changes	git add .	Stage all changes
Commit	git commit -m "message"	Save a new version
Push	git push origin main	Upload to GitHub
Pull	git pull origin main	Fetch latest changes
Log history	git log --oneline	View commit history
Create new branch	git checkout -b feature-xyz	Start a new branch
Switch branch	git checkout main	Move between branches
